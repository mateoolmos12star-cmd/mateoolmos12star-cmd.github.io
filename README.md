# Portfolio profesional - Harrison Olmos

Portafolio web personal para presentar mi perfil profesional, experiencia, habilidades, proyectos y canales de contacto.

## Sitio publicado

[Ver portfolio en GitHub Pages](https://mateoolmos12star-cmd.github.io/Porfolio/src/index.html)

## Contenido

- Perfil profesional enfocado en operaciones, soporte tecnico, automatizacion y tecnologia.
- Seccion de habilidades tecnicas y operativas.
- Proyectos destacados, incluyendo automatizacion de reportes y generacion de archivos D93 TXT.
- Experiencia profesional.
- Accesos directos a CV, GitHub, LinkedIn, correo y WhatsApp.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- GitHub Pages

Este proyecto es un sitio estatico. Para verlo localmente, abre el archivo:

```text
src/index.html
```

Tambien puedes usar la extension Live Server de Visual Studio Code para previsualizar los cambios mientras editas.
